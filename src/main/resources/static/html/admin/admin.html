What is this website about?

This website is a comprehensive platform for anime enthusiasts. It allows users to explore, manage, and keep track of various animes, genres, characters, and studios.

How do I add a new anime?

To add a new anime, go to the "Animes" section in the admin panel and click on the "Add New Anime" button. Fill in the required details, such as the anime's name, description, genres, and characters, and then click "Save."

How do I edit an existing anime?

To edit an existing anime, navigate to the "Animes" section in the admin panel. Find the anime you want to edit and click on the "Edit" button. Make the necessary changes and click "Save."
